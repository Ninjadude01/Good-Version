NINJA GAME: this is a 2d platformer game made wtih pixel art. There are orbs to collect, enemies, moving platforms, and more! You can shoot shurikens that can damage enemies by clicking the left mouse button. This is my first time making a game, and I plan to add more updates to it in the near future. There are currently 3 levels, and more will be added soon. It can be played through https://ninjadude01.itch.io/ninja-game
